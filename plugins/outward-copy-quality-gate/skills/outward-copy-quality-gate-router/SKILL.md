# Route outward copy

Treat the hook's active-turn context as the transaction contract.

1. Identify the reader, surface, desired action, factual constraints, and claims that need proof.
2. Name the copy owner in a short, non-sensitive evidence ID. Follow any project-specific owner rule; otherwise use the agent explicitly assigned to write the copy.
3. Draft or revise the copy without inventing features, results, quotes, or sources.
4. Apply `$outward-copy-quality-gate-humanizer` to the exact final copy.
5. Apply `$outward-copy-quality-gate-ogilvy` to the same final copy.
6. Append the exact privacy-safe receipt template supplied by the hook. Replace only its evidence placeholders. Do not put the prompt, copy body, credentials, names, or private context in the marker.

Do not claim a review happened when it did not. A receipt proves that the required stages were bound to this turn; it does not make an unsupported claim true.
