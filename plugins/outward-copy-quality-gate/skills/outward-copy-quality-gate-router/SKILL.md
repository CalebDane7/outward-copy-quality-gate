# Route outward copy

Use the hook's same-turn context as the routing brief. Continue the user's task.

1. Identify the reader, surface, desired action, factual constraints, and claims that need proof.
2. Follow any project-specific copy-owner rule. Otherwise, the agent assigned to the task owns the copy.
3. Draft or revise the copy without inventing features, results, quotes, or sources.
4. Apply `$outward-copy-quality-gate-humanizer` to the exact final copy.
5. Apply `$outward-copy-quality-gate-ogilvy` to the same final copy.

Do not claim a review happened when it did not. Do not add a receipt or hidden marker to the user's copy.
